Changelog.

Five phases shipped. Everything below runs on devnet; mainnet is gated on a custody signer.

phase 05 · shippedSecurity review

Security review & hardening pass

Internal review plus an independent test pass. Both found and fixed real issues: OTP replay tightened to global burn across admins, group-chat secret scrubbing added, price-impact ceiling layered on top of Jupiter slippage, mainnet start-up guard made a hard refusal.

phase 04 · shippedAudit ledger

Hash-chained audit log

Every command attempt — allowed or denied — appends to data/audit.jsonl, each entry's hash folding in the previous one. verifyAuditChain() re-walks the chain to detect any edit.

phase 03 · shippedSplitter

Atomic multi-recipient splitter

/send pays multiple recipients in one atomic transaction. Destinations resolve only to a fixed contacts file or a literal address — never a name invented from free text. Optional Groq parsing for looser phrasing, deterministic regex otherwise.

phase 02 · shippedAuto-sweep

Background auto-sweep

Token watcher detects incoming SPL across child accounts and swaps to SOL through Jupiter automatically — no command needed. /sweep forces an immediate full pass.

phase 01 · shippedFoundation

Auth & key vault

Numeric-ID allowlist, Argon2 password login, single-use TOTP, and the swappable Signer interface with a local AES-256-GCM encrypted keystore.
